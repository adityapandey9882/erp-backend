# Backend

Express + TypeScript API structure for the Company Management ERP.

## Scope

- Modular feature folders under `src/modules`
- Environment-based configuration for PostgreSQL and Supabase
- Central route composition under `src/routes`
- Ordered SQL migration placeholders under `src/database/migrations`

## Notes

- Business logic is intentionally not implemented yet
- PostgreSQL is the primary database target
- Supabase configuration is prepared for future document storage integration
